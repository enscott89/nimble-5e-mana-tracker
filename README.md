# Nimble 5E Mana Tracker

A system-agnostic manual mana tracker for 5e-style FoundryVTT games.

## Features

- Actor-flag based mana tracking.
- Token Controls launcher.
- Actor sheet and token HUD shortcuts.
- Full, half, quarter, and custom caster types.
- Manual spend/recover controls.
- Long rest recovery to maximum mana.
- Arcane Recovery helper using `ceil(caster level / 2)`.
- Dark compact tracker UI.

## Compatibility

Minimum FoundryVTT version: v12  
Verified FoundryVTT version: v14
